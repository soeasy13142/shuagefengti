74 个知名品牌首页设计预览，用于设计灵感参考。

## 品牌列表

## 格式

每个品牌一个独立 HTML 文件，命名规则 `{brand}-preview.html`。可通过浏览器直接打开查看。

## 用途

作为设计参考，了解各品牌首页的布局、配色、排版和交互风格。本项目当前采用 Claude Design 暖奶油画布风格。
