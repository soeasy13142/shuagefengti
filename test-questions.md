1. 以下哪个是JavaScript的基本数据类型？
A. Object
B. Array
C. String
D. Function
答案：C
解析：String 是基本数据类型，Object、Array、Function 是引用类型

2. 以下哪些是合法的CSS选择器？
A. .class
B. #id
C. div
D. @media
答案：ABC
解析：@media 是媒体查询，不是选择器

3. HTML中，<img>标签的alt属性用于
A. 设置图片路径
B. 设置图片宽度
C. 设置图片替代文本
D. 设置图片标题
答案：C

4. Git中，以下哪个命令用于查看提交历史？
A. git status
B. git log
C. git diff
D. git show
答案：B

5. CSS中，display:none 和 visibility:hidden 的区别是前者完全移除元素，后者只是隐藏但保留___。
答案：占位空间

6. 请解释什么是闭包。
答案：闭包是指有权访问另一个函数作用域中变量的函数，通常是在一个函数内部创建另一个函数时产生

7. TCP三次握手的过程是___、___、___。
答案：SYN,SYN-ACK,ACK

8. Vue中，父子组件通信的方式有哪些？
A. props / $emit
B. $refs
C. Vuex
D. localStorage
答案：AB

9. 以下哪个不是HTTP请求方法？
A. GET
B. POST
C. SEND
D. DELETE
答案：C

10. 什么是RESTful API？请简要说明其设计原则。
答案：RESTful API 是一种基于 HTTP 协议的 API 设计风格，核心原则包括：使用 HTTP 方法表示操作（GET查询、POST创建、PUT更新、DELETE删除），资源通过 URL 标识，无状态通信，统一接口
